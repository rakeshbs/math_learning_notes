# Math Learning Notes

Math Learning Notes is an interactive React app for building intuition in core math and machine learning topics through visual explanations.

## Topics

- Matrices
- Multivariable Calculus
- Machine Learning Math
- Deep Learning
- Reinforcement Learning
- Statistics
- Bayesian Statistics
- Activation Functions

## Tech Stack

- React 18
- Vite 5
- KaTeX / react-katex

## Getting Started

Prerequisite: Node.js 18+.

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite (typically `http://localhost:5173`).

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Create a production build
- `npm run preview` - Preview the production build locally

## Project Structure

- `src/domains/` - Domain definitions and tab configuration
- `src/*-intuition/data/` - Concepts, explanations, and details
- `src/*-intuition/visuals/` - Visual components for each concept
- `src/math-explorer/` - Shared explorer UI components
